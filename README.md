# AYBU MIS Student Login System (Dockerized)

This project demonstrates how to design a student login system using a microservice architecture.

## Features
- [cite_start]**Backend:** Bottle (Python 3.9) [citation: 1]
- **Database:** PostgreSQL 13
- [cite_start]**Web Server:** Nginx (Reverse Proxy) [citation: 2]

- [cite_start]**Libraries:** SQLAlchemy, Art, Psycopg2 [cite: 4]
